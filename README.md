### Команды:
- `npm run install` - установка необходимых библиотек
- `npm run dev` - запуск проекта в режиме разработчика
- `npm run build` - сборка проекта
- `npm run start` - запуск проекта (локально)

### Доступные страницы:
- `/signin` - вход
- `/signup` - регистрация
- `/chat` - чат
- `/profile` - профиль
- `/error` - ошибка
- `/edit` - редактор
